POWER BI


Power BI Desktop es una aplicación gratuita
que se instala en una computadora personal
y permite conectar datos, transformarlos y
visualizarlos.


Los reportes desarrollados con Power BI Desktop pueden compartirse con otras personas usando Power BI Service.

Power BI Service es un servicio en la nube administrado por Microsoft que permite a los usuarios ver dashboards, reportes y aplicaciones de Power BI usando un navegador web o aplicaciones para móviles Windows, iOS o Android.

Oportunidades Power BI

La herramienta es muy intuitiva y amigable con el usuario. Cualquiera que haya tenido un acercamiento con las herramientas básicas de Microsoft Office podrá hacer dashboards desde el primer día que lo maneje.

También tiene una amplia compatibilidad con diferentes fuentes de información, desde archivos CSV, Excel y Bases de datos, o incluso desde fuentes externas como Facebook, analytics o incluso cualquier tabla que aparezca en una pagina web con solo tener el enlace de esta. Todos esos datos pueden ser tratados en PBI sin manipular el origen de la información, de tal forma que pueda ser concentrada y presentada en un solo reporte.

Cuenta con un lenguaje propio para transformación de datos llamado DAX, sencillo de aprender para cualquiera que conozca lógica de programación.








Beneficios con Power BI


La interfaz es muy parecida a la de las herramientas de Microsoft Office. Para crear dashboards basta con importar la información desde cualquier fuente compatible y arrastrar los campos al objeto visual que se desee presentar (una tabla o un gráfico).

Al ver un reporte en PBI, el usuario tendá una experiencia parecida a Excel, pero al estar sobre el servicio en la nube, podrá manejar mas información y el procesamiento de datos no dependerá de la capacidad de su equipo de computo.
